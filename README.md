# Bright-motors-sales-analysis
A data analysis project exploring Bright Motors vehicle sales to identify trends, pricing patterns, model performance, and key factors influencing selling price using fields like year, make, model, condition, and odometer.

## Objectives
The main objectives of this analysis are to:

1. Identify the top-selling vehicle makes, models. 
2. Analyze sales trends over time to detect seasonal patterns.  
3. Examine regional sales distribution to highlight high- and low-performing areas.  
4. Explore price ranges and vehicle conditions to understand customer preferences.  
5. Provide actionable insights that can guide marketing, inventory, and pricing strategies.

## Dataset
The dataset includes vehicle information such as year, make, model, trim, body type, transmission, VIN, condition, odometer reading, color, interior, seller, market reference value (MMR), selling price, and sale date. The data was cleaned to ensure accuracy and consistency.(Bright cars csv file).

## Tools and Technologies
- **Excel** – Data cleaning, analysis, and visualizations  
- **SQL** – Data querying  
- **Miro** – Project planning   
- **Canva** – Designing presentations and visual content

## Analysis Steps
1. Planned the project and defined objectives.  
2. Imported the dataset into **SQL** and performed queries to extract and filter relevant data.  
3. Exported the queried data to **Excel** for analysis using pivot tables and charts.  
4. Created visualizations in **Excel** to summarize findings.  
5. Transferred visualizations to **Canva** for the final presentation.

## Key Findings
- Top-performing vehicle models and makes  
- Seasonal and monthly sales patterns  
- Regional sales distribution  
- Insights into price ranges and vehicle conditions
